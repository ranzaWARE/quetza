Whisper è un modello AI open-source per la trascrizione audio. Deve essere installato sul server Docker come container separato o nel PATH del container Node.

Setup iniziale: il container quetza-whisper scarica i modelli al primo avvio (~1.5GB). Per la diarizzazione serve un token HuggingFace gratuito — crea account su huggingface.co e accetta la licenza del modello pyannote/speaker-diarization-3.1.
Export globale
Scarica un archivio ZIP con tutte le note e gli audio di tutti gli utenti. Utile per migrare l'istanza su un altro server.
Scarica archivio completo
Import globale
Carica un archivio esportato da un'altra istanza. Gli utenti mancanti vengono creati con password temporanea changeme123.
Nota: l'import globale è additivo — non elimina i dati esistenti. Le note con lo stesso ID vengono aggiornate. Gli utenti già esistenti non vengono toccati.
